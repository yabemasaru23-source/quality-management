# 神田マーケティング集客ノート（クオリティマネジメントHP用）

このHPは「神田昌典の手法で集客する」ことを設計思想にする。ネット調査でまとめた要点を、このフォルダに蓄積していく。
※ スキル `kanda-marketing` と同じ内容を、このHP用に運用メモ化したもの。新しい学びはここに追記していく。

---

## このHPに効いている神田の原則（実装済み）

1. **売り込まない2ステップ集客** … 一度の訪問で契約させない。無料の入口（壁打ち・セミナー・LINE）で手を挙げてもらい、メルマガ・noteで教育してから提案する。
2. **新PASONAの法則でページを構成** … Problem→Affinity→Solution→Offer→Narrow→Action の順で読者の感情を動かす。
3. **エモーショナルマーケティング** … スペックでなく「手に入る未来」を描く。日常言語で書く。五感に訴える。
4. **見込み客の教育** … 情報発信ボタン（時間創造部マーケティング・メルマガ・note）で継続接触し、信頼を積む。

## 新PASONA テンプレ（コピーを書くときの型）

| 順 | 要素 | このHPでの当てはめ |
|---|------|------|
| P | Problem | 「人手不足・同じ作業の繰り返し・AIが続かない」 |
| A | Affinity | 「私たちも同じでした」と寄り添う |
| So | Solution | AIエージェント導入＋感動マーケで解決 |
| O | Offer | 初回壁打ち無料／セミナー／顧問 |
| N | Narrow | 毎月の受付社数を絞る（数字は確定後） |
| A | Action | LINE登録・壁打ち予約ボタン |

## エモーショナル・チェック（公開前に毎回）

- [ ] スペックでなく「手に入る未来」を描いたか
- [ ] 顧客の日常言語か（×業務効率化 ○ムダな残業が消える）
- [ ] 無料の入口→教育→提案の2ステップになっているか
- [ ] USP（なぜ“やべまさる”から買うのか）が一言で言えるか
- [ ] お客様の声・実績は“数字”で（偽数字は禁止＝第10条）
- [ ] 限定・緊急（人数/期限）はあるか（嘘の締切は禁止）
